take the linear instrumental variables equation y_i = \beta_0 + \beta_1 x_i + \beta_2 w_i + e_i, where z_i is an instrument

You are then told that Wi is unavailable. Consider the 2SLS estimator of \beta_1 obtained from the misspecified IV regression, by regressing yi on xi only, using zi as an instrument for xi.

a) Derive the conditions under which your IV estimate of \beta_1 from the misspecified model is consistent.

b) Find the asymptotic distribution of this IV estimate in part a) as n \rightarrow \infin.

A3 Q4 PART-A : THE MISSPECIFIED MODEL

if we use z as an instrument for x, and without other regressors, then we get this

\hat{\beta}_{TSLS} \xrightarrow{p} \dfrac{cov(z,y)}{cov(z,x)} = \dfrac{cov(z,\beta_0 + \beta_1 x + \beta_2 w + e)}{cov(z,x)} = \dfrac{cov(z,\beta_1 x)}{cov(z,x)} + \dfrac{cov(z,\beta_2 w)}{cov(z,x)} + \dfrac{cov(z,e)}{cov(z,x)} = \beta_1 + \beta_2 \cdot \dfrac{cov(z, w)}{cov(z,x)} + \dfrac{cov(z,e)}{cov(z,x)}

\hat{\beta}_{TSLS} \xrightarrow{p} \beta_1 + \beta_2 \cdot \dfrac{cov(z, w)}{cov(z,x)} + 0

so if \text{corr}(z,w) \ne 0 and \beta_2 \ne 0 , the estimate is biased, and inconsistent

if \text{corr}(z,w) = 0 , it is consistent

and if \beta_2 = 0 , it is consistent, but our model is mispecified (in spirit, if we really wanted to include w, that is)

A3 Q4 PART-B : THE ASYMPTOTIC DISTRIBUTION

\hat{\beta}_{IV} = \beta_1 + \beta_2 \cdot \dfrac{ \frac{1}{n} \sum (z-\mu_z)(w-\mu_w) }{ \frac{1}{n} \sum (z-\mu_z)(x-\mu_x) } + \cfrac{ \frac{1}{n} \sum (z-\mu_z) e }{ \frac{1}{n} \sum (z-\mu_z)(x-\mu_x) }

\hat{\beta}_{IV} = \beta_1 + \beta_2 \cdot \dfrac{ \frac{1}{n} \sum h }{ \frac{1}{n} \sum r } + \cfrac{ \frac{1}{n} \sum q }{ \frac{1}{n} \sum r }

\hat{\beta}_{IV} = \beta_1 + \beta_2 \cdot \dfrac{ \bar{h} }{ \bar{r} } + \cfrac{ \bar{q} }{ \bar{r} } [this is the style of decomposition from SW that I thought was useful]

\bar{q} \backsim N \big ( 0, \frac{\sigma_q^2}{n} \big ), and \bar{r} \backsim N \big ( \mu_r, \frac{\sigma_r^2}{n} \big ) which makes \dfrac{ \bar{q} }{ \bar{r} } normally distributed with mean 0 and variance \dfrac{1}{n} \cdot \dfrac{var((z-\mu_z)e)}{[cov(x,z)]^2}

if \beta_2 = 0 or cov(z,w) = 0 then that would be the variance of the estimator we're looking at, too

anyhow, the term \dfrac{ \bar{h} }{ \bar{r} } is a pickle

\bar{r} \backsim N \big ( \mu_r, \frac{\sigma_r^2}{n} \big ) so (\bar{r} - \mu_r) \backsim N \big ( 0, \frac{\sigma_r^2}{n} \big ) and we do the same thing for \bar{h}

\dfrac{ \bar{h} }{ \bar{r} } = \dfrac{ (\bar{h} - \mu_h) + \mu_h }{ (\bar{r}-\mu_r)+\mu_r } = \dfrac{ (\bar{h} - \mu_h) }{ (\bar{r}-\mu_r)+\mu_r } + \dfrac{ \mu_h }{ (\bar{r}-\mu_r)+\mu_r } is a pickle

i think that if corr(x,w) \ne 0, then \frac{ \bar{h} }{ \bar{r} } is more complex, as numerator and denominator are normal random variables that are correlated

even if that doesn't influence whether or not the estimator is consistent, it will make the distribution complex and generally non-normal

if corr(x,w) = 0, then it is still hard

the 2 random variables, \bar{h} and \bar{r} will not be correlated, but it is still a ratio of 2 normal variables
