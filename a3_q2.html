RSS^{OLS} = \sum \hat{e}^2 = \sum (y - x \hat{\beta})

under ols, var(y) = var(\hat{y} + \hat{e}) = var(\hat{y}) + var(\hat{e}), since cov(\hat{y},\hat{e}) = 0

so RSS^{OLS} \lt TSS

but for IV

RSS^{IV} = \sum \tilde{e}^2 = \sum (y - x \tilde{\beta})

\tilde{\beta} did not come from using x, but rather from \hat{x} , so it is possible the RSS^{IV} \gt TSS

so it is possible that RSS^{IV} \gt RSS^{OLS}

so in that sense, IV will not necessarily fit better than OLS

but obviously, that's not the point. IV is for estimating the parameters of the structural equation, so why would the R2 matter
