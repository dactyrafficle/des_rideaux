a3 q5 - part a 


X^2 should be treated as endogenous

from the conditioning theorem
E[g(X)e] = E[ E[g(X)e|X] ] = E[ g(X) E[e|X] ]

therefore 
E[X^2e] = E[ E[ X^2 e|X] ] = E[ X^2 E[e|X] ]

and noting that E[e|X] \ne 0 because
E[Xe] \ne 0
E[Xe] = E [ E[Xe|X] ] = E [ X E[e|X] ] \ne 0
which only works if E[e|X] \ne 0

so, therefore, since E[e|X] \ne 0, E[X^2e] \ne 0, so X^2 is endogenous

a3 q5 - part b

we should look at X and X^2 as 2 separate regressors. so we have 2 endogenous regressors, and thus we need at least 2 valid instruments

i think that by suggesting the use of (1, Z, Z^2) as instruments, what is being asked is that we use Z and Z^2 as instruments for X and X^2

the first-stage equations become

  • X = \gamma_0 + \gamma_1 Z + \gamma_2 Z^2 + u

  • X^2 = \delta_0 + \delta_1 Z + \delta_2 Z^2 + v

the result of the first-stage is :

  • \hat{X} = \hat{\gamma}_0 + \hat{\gamma}_1 Z + \hat{\gamma}_2 Z^2

  • \hat{X}^2 = \hat{\delta}_0 + \hat{\delta}_1 Z + \hat{\delta}_2 Z^2

the model is just-identified, since we have m=2 endogenous regressors, and k=2 valid instruments
